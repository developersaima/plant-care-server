# 🌿 PlantCare - Server

## 📖 Description

The PlantCare Server is the backend of the PlantCare application, built with **Node.js**, **Express.js**, **TypeScript**, and **MongoDB**. It provides RESTful APIs for managing plant data, handling user requests, and connecting the frontend with the database.

---

## 🎯 Project Purpose

The purpose of this backend is to provide a secure and scalable API that supports plant management operations. It enables seamless communication between the client application and the MongoDB database while maintaining a clean and organized server architecture.

---

## ✨ Key Features

- RESTful API Development
- MongoDB Database Integration
- Plant CRUD Operations
- Fetch All Plants
- Fetch Single Plant Details
- Add New Plant
- Delete Plant
- CORS Configuration
- Environment Variable Support
- TypeScript-Based Backend

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- TypeScript
- MongoDB
- CORS
- Dotenv
- JOSE
- Zod

---

## 🔄 Workflow

- Receives requests from the client application.
- Processes API requests using Express.js.
- Connects to MongoDB to store and retrieve plant data.
- Returns structured JSON responses to the frontend.
- Handles validation and server-side logic.
- Provides secure communication between the frontend and the database.
